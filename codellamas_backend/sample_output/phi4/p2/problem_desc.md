#### Problem Description and Learning Objectives

**Domain:** Library Management System

**Objective:**

- Identify the "Duplicate Method" code smell in a Java Spring Boot application.
- Refactor the code safely without altering its observable behavior.
- Understand existing functionality through JUnit 5 test cases.
- Improve code readability, maintainability, and structure.

**Expected Task for Students:**

1. Review the provided codebase containing duplicate methods.
2. Run and analyze the given test cases to understand intended behavior.
3. Refactor the code by removing duplicates while ensuring all tests pass post-refactoring.
